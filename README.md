# Usage

Create a new project based on template.

`npx create-simple-projects`

Or

`npm i create-simple-projects - g && npx csp`

## Options

### template

Select template project, Available options:

- `ts`

`npx csp --template ts`

### name

Name for new project.

`npx csp --name test`
